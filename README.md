## Information We Collect

WeeklyQuest collects **limited user-provided data** strictly to enable core app functionality.

This may include:

* **Self-reported wellness information**, such as mood, sleep, hydration, gut status, and focus activity
* **In-app progress data**, such as quests, XP, achievements, streaks, and titles
* **User-generated content**, such as chat messages and reactions

This data is used **only** to operate app features and provide a personalized experience. It is not used for advertising, tracking, or profiling.

## Authentication

WeeklyQuest uses **anonymous authentication** provided by Firebase.

* No email address, real name, phone number, or personal identifier is required
* Each user is assigned a **randomly generated identifier** used only within the app
* This identifier cannot be used to identify you outside of WeeklyQuest

## Data Storage & Transmission

* **Primary gameplay and wellness data** is stored locally on your device
* Certain features (such as chat and profile synchronization) securely transmit data to backend services
* All data transmission uses encrypted connections

WeeklyQuest does **not** sell user data, share data for advertising, or track users across apps or websites.
